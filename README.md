<h1>Aqua-Analytics : An Insight into India's Water Accessibility</h1>
<h3>Predicting Implementation Integrity for India's Water Mission</h3>
<h4>This project analyzes the current state of water access and uses AI to predict future challenges in the certification and implementation of the "Har Ghar Jal" mission, identifying Indian states and union territories(UT), that require proactive support.</h4>
